# Acceptance Tests

Acceptance tests are documented as user-facing scenarios in:

- `docs/testing/acceptance-test-results.md`

Primary flows:

- Pelapor creates and tracks a report.
- Admin reviews, prioritizes, and assigns a report.
- Technician updates progress to resolved.
- Pelapor confirms or rejects the result.
- Manager views aggregate dashboard data.

Production smoke checks have been run through the public Worker URL. Full visual evidence screenshots can be added under `evidence/screenshots/` during the final review session.
